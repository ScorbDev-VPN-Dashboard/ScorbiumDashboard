## Требования

- **OS**: Ubuntu 20.04+ / Debian 11+
- **Docker**: 24.0+
- **Docker Compose**: v2.0+
- **RAM**: минимум 512 MB
- **Домен**: с A-записью на IP сервера (для SSL)
