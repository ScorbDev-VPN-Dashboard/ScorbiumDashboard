
# PasarGuard / Marzban

VPN Dashboard интегрируется с Marzban через REST API.

## Настройка подключения

В `.env`:
```env
PASARGUARD_ADMIN_PANEL=https://your-panel.com:8012
PASARGUARD_ADMIN_LOGIN=admin
PASARGUARD_ADMIN_PASSWORD=your_password
PASARGUARD_API_KEY=   # оставьте пустым
```

> 💡 Используйте логин/пароль. API ключ необязателен и имеет срок действия.

## Проверка подключения

В панели: **PasarGuard → Статус → Проверить**

Или через API:
```bash
curl https://your-domain.com/api/v1/healthy
```

## Группы VPN

Группы определяют к каким серверам/протоколам будет иметь доступ пользователь.

### Настройка групп

1. Панель → **PasarGuard → Группы**
2. Нажмите «Загрузить» для получения групп из Marzban
3. Выберите нужные группы
4. Сохраните

Или в боте: `/admin` → **🌐 Группы VPN**

### Как работает

При создании подписки система автоматически назначает выбранные группы пользователю в Marzban.

## Автоматические задачи

| Задача | Интервал | Описание |
|--------|----------|----------|
| Истечение ключей | 5 минут | Помечает просроченные ключи |
| Синхронизация | 1 час | Синхронизирует статусы с Marzban |
| Уведомления | 5 минут | Отправляет предупреждения за 3 дня |
| Автопродление | 5 минут | Списывает с баланса и продлевает |

## Формат ключей

Ключи создаются с именем `vpn_{user_id}_{key_id}` в Marzban.

Ссылка подписки: `https://your-panel.com/sub/{token}`

## Синхронизация вручную

В панели: **PasarGuard → Быстрые действия → Синхронизировать ключи**

Или через API:
```bash
curl -X POST https://your-domain.com/panel/vpn/sync \
  -H "Cookie: vpn_session=your_token"
```
