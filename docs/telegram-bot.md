
# Telegram-бот

## Создание бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`
3. Введите название и username бота
4. Скопируйте токен и вставьте в `.env` → `TELEGRAM_BOT_TOKEN`

## Режимы работы

### Long Polling (для разработки)
```env
TELEGRAM_TYPE_PROTOCOL=long
```
Бот сам опрашивает Telegram. Не требует домена.

### Webhook (для продакшена)
```env
TELEGRAM_TYPE_PROTOCOL=webhook
TELEGRAM_WEBHOOK_URL=https://your-domain.com/webhook/bot
```
Telegram отправляет обновления на ваш сервер. Требует HTTPS.

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню |
| `/profile` или `/me` | Профиль пользователя |
| `/status` | Статус подписок |
| `/ping` | Статус серверов |
| `/top` | Топ рефереров |
| `/gift @username` | Подарить подписку |
| `/keys` | Список активных подписок |
| `/id` | Узнать свой Telegram ID |
| `/admin` | Панель администратора (только для админов) |

## Настройка через панель

Все настройки бота доступны в панели: **Telegram → вкладки**

### Вкладка «Бот»
- Включить/выключить бота
- Прямая отправка сообщений пользователям
- Настройка реферальных бонусов

### Вкладка «Кастомизация»
- Приветственное сообщение (поддерживает `{name}`)
- Системные сообщения (бан, отключение, выдача ключа)
- Текст раздела «О проекте»

### Вкладка «Кнопки»
Визуальный редактор клавиатуры с превью:
- Drag & drop кнопок
- Максимум 3 кнопки в ряду
- Выбор цвета: 🟢 success / 🔵 primary / 🔴 danger
- Кнопка «разбить ряд» для разделения

### Вкладка «Фото»
Загрузка фото для каждого раздела бота:
- Приветствие, Купить, Мои подписки, Баланс, О проекте, Поддержка, Профиль

### Вкладка «Mini App»
Кнопка «🖥 Открыть панель» в `/admin` — открывает панель прямо в Telegram.
Требует HTTPS домен.

### Вкладка «Канал»
Обязательная подписка на канал перед использованием бота.
Бот должен быть администратором канала.

## Антиспам

Встроенный rate limiter:
- 10 токенов на пользователя
- Пополнение 5 токенов/сек
- Блокировка на 30 сек при исчерпании

## Уведомления

Бот автоматически отправляет:
- ⚠️ За 3 дня до истечения подписки
- ✅ После успешной оплаты
- 🔄 При автопродлении с баланса
- 💬 Ответы на тикеты поддержки
