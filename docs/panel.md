
# Панель управления

Веб-панель доступна по адресу `https://your-domain.com/panel/`

## Разделы панели

### 📊 Дашборд
- Статистика: пользователи, подписки, выручка, тикеты
- График выручки за 7 дней
- Статистика Marzban (онлайн, трафик, RAM, CPU)
- Последние пользователи и платежи

### 👥 Пользователи
- Список всех пользователей
- Поиск по имени/username
- Управление: бан/разбан, пополнение баланса
- Подарить подписку
- История платежей и ключей

### 📋 Тарифы
- Создание тарифных планов
- Название, цена, количество дней
- Включить/выключить тариф

### 🔑 Подписки
- Все VPN-ключи пользователей
- Продление, отзыв
- Создание вручную (подарок)
- Синхронизация с Marzban

### 💳 Платежи
- История всех платежей
- Фильтр по статусу
- Возврат платежа

### 🎁 Промокоды
- Создание промокодов
- Типы: скидка %, баланс ₽, дни к подписке
- Ограничение использований

### 👥 Рефералы
- Статистика реферальной программы
- Топ рефереров

### 💬 Поддержка
- Тикет-система
- Ответы пользователям
- Статусы: открыт / в работе / закрыт
- Приоритеты: низкий / средний / высокий

### 📢 Рассылки
- Создание рассылок
- Цели: все / активные / истекшие
- HTML форматирование

### 🌐 PasarGuard
- Статус подключения к Marzban
- Пользователи VPN
- Группы и ноды
- Быстрые действия

### ⚙️ Telegram
Вкладки: Бот / Кастомизация / Кнопки / Фото / Mini App / Канал

### 💾 Резервная копия
- Экспорт БД (SQL / .gz)
- Импорт из файла

## Авторизация

Панель защищена cookie-сессией. Логин/пароль задаются в `.env`:
```env
WEB_SUPERADMIN_USERNAME=admin
WEB_SUPERADMIN_PASSWORD=your_password
```

## Безопасность

- Rate limiting: 120 запросов/мин для панели, 60 для API
- Сессия действует 24 часа
- Нельзя забанить самого себя или другого администратора
