Очистка базы данных

Для очистки базы данных используй команды в терминале сервера. Перед очисткой сделай резервную копию.

Очистить все таблицы (сохранить структуру)
psql $DATABASE_URL -c " TRUNCATE users, subscriptions, payments, vpn_keys, plans, promos, referrals, support_tickets, broadcasts, bot_settings RESTART IDENTITY CASCADE;"
Полный сброс (drop + migrate)
psql $DATABASE_URL -c "DROP SCHEMA public CASCADE; CREATE SCHEMA public;" alembic upgrade head
Удалить только платежи и подписки
psql $DATABASE_URL -c " TRUNCATE payments, subscriptions, vpn_keys RESTART IDENTITY CASCADE;"
Через Docker Compose
docker compose exec db psql \ -U $DB_USER -d $DB_NAME \ -c "TRUNCATE users CASCADE;"
