# VPN Dashboard

Dashboard for managing VPN subscriptions, users, and payments via Telegram bot.

## Features

- 🤖 Telegram bot integration
- 💳 Yookassa payment processing
- 🔐 Pasarguard VPN management
- 📊 Database management (PostgreSQL)
- 📝 Configurable logging system

## Tech Stack

- Python 3.10+
- PostgreSQL
- Pydantic Settings
- Loguru
- SQLAlchemy (async)
